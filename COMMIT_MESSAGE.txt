This PR implements a complete documentation system using MkDocs Material,
Doxygen, and GitHub Actions for automatic deployment to GitHub Pages.

## What's Added

### Infrastructure
- GitHub Actions workflow for auto-deployment
- MkDocs configuration with Material theme

### Documentation Content
- Professional homepage with features and examples

- Examples index with learning paths
- Setup and deployment guides

### Features
✅ Auto-deploys on every commit to master
✅ Searchable API reference from Doxygen

✅ Live reload during local development
✅ Zero maintenance after setup

## Deployment Instructions

1. Enable GitHub Pages:
   - Go to Settings → Pages
   - Source: "Deploy from a branch"
   - Branch: "gh-pages" / (root)

2. Apply Doxyfile changes:
   - See docs/Doxyfile.patch for required updates

3. Merge and wait:
   - Actions workflow runs automatically
   - Site live at https://iosonata.github.io/IOsonata/
